═══ 使用说明 ═══

▼ 开关差异
1. 顶部选「左侧 / 右侧」两个预设 → 点「对比」
2. 列表里直接看哪个开关变了，红色 = 关 / 绿色 = 开
3. 点条目右侧「同步到右」→ 把左侧开关状态同步过去
4. 全部改完 → 点「覆盖保存到右」

▼ 条目迁移
1. 顶部选好左/右预设并对比
2. 在列表里找到要迁移的条目 → 点「📦 迁移」加入待办队列
3. 想改插入位置 → 点队列里那条的「📍 位置」自己选放在哪条之后
4. 队列里都准备好 → 点最右的「✨ 应用迁移」一键覆盖保存
